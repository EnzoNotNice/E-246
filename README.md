---

## المميزات الرئيسية

* **إدارة السيرفر:** حظر، طرد، تحذيرات، إسكات مؤقت، وقفل/فتح الرومات.
* **نظام الحماية (AutoMod):** مضاد للروابط، مضاد للسبام، والتحقق البشري (Captcha).
* **نظام المستويات (XP):** احتساب نقاط التفاعل الصوتي والكتابي بدقة ورسم بطاقات رتب حديثة.
* **نظام التذاكر (Tickets):** إنشاء تذاكر الدعم الفني وتصنيفها مع حفظ المحادثات.
* **تتبع الدعوات (Invites):** حساب تفصيلي للدعوات (الحقيقية، المزيفة، الإضافية، المغادرين).
* **الترحيب والمغادرة:** إرسال رسائل ترحيب مخصصة بالخاص أو العام مع صور وبطاقات دخول.
* **الأتمتة (Automation):** تفعيل الفاصل التلقائي (Auto-line) والضريبة التلقائية لبروبوت.
* **لوحة تحكم ويب (Web Dashboard):** واجهة ويب متجاوبة بالكامل مع الجوال للتحكم بجميع الإعدادات.

## قائمة الأقسام والأوامر

| القسم | الأوامر | الوصف |
| :--- | :--- | :--- |
| **الإدارة** | `/ban`, `/kick`, `/timeout`, `/warn`, `/clear`, `/lock` | ضبط وانضباط السيرفر ومعاقبة المخالفين |
| **المستويات** | `/rank`, `/top`, `/leaderboard`, `/settings` | نظام النقاط والترقيات الصوتي والكتابي |
| **التذاكر** | `/ticket-setup`, `/send-panel`, `/open`, `/close` | الدعم الفني وإدارة بطاقات المساعدة |
| **الدعوات** | `/invites`, `/info`, `/addrank`, `/setlogs` | تتبع إحصائيات دعوات الأعضاء |
| **الأتمتة** | `/autoline`, `/autotax`, `/setline`, `/tax` | تشغيل الفواصل والضريبة تلقائياً بالرومات |
| **الحماية** | `/setup`, `/setlimits`, `/whitelist` | إعداد حدود العمليات والحماية من التخريب |
| **العامة** | `/user`, `/avatar`, `/banner`, `/server`, `/help` | معلومات الأعضاء والسيرفر والملفات الشخصية |

## متطلبات التشغيل والبدء السريع

1. **تثبيت الحزم المطلوبة:**
   ```bash
   npm install
   ```

2. **تهيئة ملف الإعدادات (`.env`):**
   قم بإنشاء ملف `.env` في المسار الرئيسي بالبيانات التالية:
   ```env
   DISCORD_TOKEN=توكن_البوت
   CLIENT_ID=معرف_البوت
   GUILD_ID=معرف_السيرفر_للتجربة
   CLIENT_SECRET=سيكرت البوت
   CALLBACK_URL=http://localhost:3000/auth/callback
   ```

3. **تشغيل البوت :**
   ```bash
   node index.js
   ```
